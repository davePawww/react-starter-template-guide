Vercel Setup

Why Vercel?

Vercel offers the best developer experience for deploying Vite apps. Zero-config, global CDN, automatic SSL, and instant rollbacks. Made by the creators of Vite!

Connect GitHub to Vercel

  1. Go to vercel.com and sign up
  2. Click Add New... ‚Üí Project
  3. Import your GitHub repository
  4. Vercel auto-detects Vite settings - click Deploy
