Installation Guide

Step-by-step instructions to set up your React project with Vite, TypeScript, and Tailwind CSS.

Step 1: Create Vite + React TypeScript Project

Why Vite?

Vite provides instant server startup and lightning-fast Hot Module Replacement (HMR). It's significantly faster than Create React App and has native TypeScript support.

bash 
# Create new project
pnpm create vite@latest my-app --template react-ts

# Navigate to project
cd my-app

# Install dependencies
pnpm install
ğŸ’¡ Tip

Replace my-app with your desired project name.

Step 2: Install Tailwind CSS

Why Tailwind CSS?

Tailwind is a utility-first CSS framework that enables rapid UI development. Instead of writing custom CSS, you use pre-built utility classes directly in your HTML/JSX. It also provides a consistent design system out of the box.

bash 
# Install Tailwind and dependencies
pnpm add -D tailwindcss postcss autoprefixer

# Initialize Tailwind config
pnpm exec tailwindcss init -p

Configure tailwind.config.js

tailwind.config.js 
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

Add Tailwind to CSS

src/index.css 
@tailwind base;
@tailwind components;
@tailwind utilities;
âš ï¸ Important

Make sure to add the Tailwind directives at the top of your index.css file. Delete any existing styles that might conflict.

Step 3: Install ShadCN UI

Why ShadCN UI?

ShadCN UI provides professionally designed, accessible components that you fully own and can customize. Unlike traditional component libraries, the components are copied into your project, giving you complete control. Built on Radix UI primitives for full accessibility.

bash 
# Initialize ShadCN in your project
pnpm dlx shadcn@latest init

Follow the prompts. Recommended settings:

  • Style: Default
  • Base color: Slate
  • CSS variables: Yes

Add Components

bash 
# Add common components
pnpm dlx shadcn@latest add button
pnpm dlx shadcn@latest add card
pnpm dlx shadcn@latest add input
pnpm dlx shadcn@latest add label
pnpm dlx shadcn@latest add accordion
pnpm dlx shadcn@latest add dialog
pnpm dlx shadcn@latest add dropdown-menu
pnpm dlx shadcn@latest add tabs
